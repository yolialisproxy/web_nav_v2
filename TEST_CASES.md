## 📝 测试执行说明

### 测试环境要求
- 操作系统: Windows 10+/macOS 12+/Ubuntu 22.04+
- 屏幕分辨率: 1920x1080 及 移动端分辨率
- 网络环境: 正常宽带 / 弱网模拟

### 验收通过标准
1. 所有 P0 用例 100% 通过
2. P1 用例通过率 ≥ 90%
3. 无崩溃级、数据丢失级bug
4. 核心功能无功能性阻塞问题
5. 性能指标符合要求

### Bug 等级定义
- 🟥 阻塞: 核心功能不可用，无法继续测试
- 🟧 严重: 主要功能异常，影响正常使用
- 🟨 一般: 次要功能异常，不影响主流程
- 🟩 轻微: 样式/文案/体验优化问题
