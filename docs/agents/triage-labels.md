# Triage Labels

The skills speak in terms of five canonical triage roles. This file maps those roles to the actual label strings used in this repo's issue tracker.

| Label in skills | Label used here | Meaning |
| --------------- | --------------- | ------- |
| `needs-triage` | `needs-triage` / `需要评估` | Maintainer needs to evaluate this issue |
| `needs-info` | `needs-info` / `需要信息` | Waiting on reporter for more information |
| `ready-for-agent` | `ready-for-agent` / `可执行` | Fully specified, ready for an AFK agent |
| `ready-for-human` | `ready-for-human` / `需人工` | Requires human implementation |
| `wontfix` | `wontfix` / `不修复` | Will not be actioned |

When a skill mentions a role (e.g. "apply the AFK-ready triage label"), use the corresponding label string from this table. Both English and Chinese labels are accepted — use whichever fits your workflow.

Edit the right-hand column to match whatever vocabulary you actually use.
