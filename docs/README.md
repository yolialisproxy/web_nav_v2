## 🎯 开发顺序
严格按照以下顺序实现，不要跳步：

```
1.  index.html 骨架 + CSS 核心变量
2.  侧边栏菜单系统 (Menu_System.md) ✅ 这部分做不对其他都免谈
3.  状态机 + 主题系统
4.  数据加载与索引
5.  卡片渲染
6.  搜索系统
7.  触觉交互细节 (Haptic_Feel.md)
8.  隐藏功能 (Missing_Features.md)
9.  最终对照 Acceptance_Criteria.md 逐项验收
```
