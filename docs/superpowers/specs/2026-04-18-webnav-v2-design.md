# 啃魂导航网站 V2 设计规范
## 2026-04-18

## 一、项目概述

啃魂导航网站是一个高质量、零依赖、零配置的纯静态导航站。基于已收集的超过 8,700 个专业网站分类数据。

## 二、架构设计

### 技术栈
- **数据层**: `websites.json` 单文件原子写入，自动备份，版本化
- **业务层**: 纯 Python 后端，零外部依赖，标准库实现
- **表示层**: 单页 HTML + 原生 JavaScript
- **交互层**: 纯 CSS3 动画，无任何前端框架 (无 React/Vue/jQuery)

### 核心原则
1. ✅ 零构建、零编译、直接打开即可运行
2. ✅ 所有交互 0 延迟
3. ✅ 完整响应式适配手机/平板/桌面
4. ✅ 支持键盘导航
5. ✅ 深色主题 + 金色点缀色

## 三、界面布局

```
┌─────────────────────────────────────────────────────────┐
│ 🔍 顶部搜索栏 + 分类导航菜单                            │
├─────────┬───────────────────────────────────────────────┤
│         │  📊 9 个主分类卡片 (3x3 网格)                   │
│         ├───────────────────────────────────────────────┤
│ 侧边栏  │  📋 当前分类子分类列表                          │
│ 可折叠  ├───────────────────────────────────────────────┤
│         │  🔗 网站卡片列表，无限滚动                       │
└─────────┴───────────────────────────────────────────────┘
```

### 组件分解
1. 顶部搜索栏：全局搜索 + 分类切换
2. 主分类网格：9 大分类卡片展示
3. 侧边栏导航：分类树折叠展开
4. 网站卡片列表：卡片式布局，支持过滤排序
5. 页脚：关于、统计、版权信息

- ✅ 所有分类正确显示
- ✅ 搜索响应 < 50ms
- ✅ 移动端完美适配
- ✅ 无控制台报错
- ✅ 0 外部依赖
